# Vor dem Spiel

Vor dem Spiel erschafft jeder Spieler seinen Charakter, dies geschieht gemeinsam, damit alle Charaktere gut zusammenpassen. Jeder Charakter wird dabei auf dem Charakterbogen festgehalten.
